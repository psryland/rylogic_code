Swizzle - Quick Reference

'Swizzling' means to rearrange the elements of a container.
  • Use alphabet characters to define the mapping from source to output.
  • Changes in character case between source and output mapping cause changes in case in the output.
  • Mappings may contain whitespace characters (which are not included in the mapping)
  • Characters can be repeated to define blocks, but they must be contiguous.
  • If a source block is longer than an output block, then a subset of the source block is mapped to the output
  • If a source block is shorter than an output block, then characters from the source block are repeated to fill the output block
