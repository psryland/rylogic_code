Main Window
Log View
This region displays the contents of the loaded or streamed log file data. Scroll this view using the mouse wheel, page up/down keys, or up/down arrow keys. Select rows by holding down the left mouse button and dragging, or hold down the Shift or Control key while selecting.
Right clicking in this view displays a context menu. The menu items are:
Option Function
Select All Select all rows in the log file.
Copy Copy the selected rows to the clip board.
Clear Log Attempts to reset the contents of the log file.
Highlight Row...
Filter Row...
Transform Row...
Action Row...
 Opens the highlighting, filtering, text transforms, or click actions pattern editor, initialised with data from the selected line.
Find Next
Find Previous
 Jump to the next or previous line in the log file that matches the last search pattern.
Toggle Bookmark Add or remove a bookmark from the selected line.
Tool Bar
The tool bar contains buttons that allow quick navigation to, and control of, various features of RyLogViewer. In order from left to right, these are:
Button Function
Open File Choose a log file to be displayed.
Refresh Update the display by forcing a reload of data from the currently loaded log file.
Quick Filter Only show rows that are highlighted.
Highlighting Left click to enable or disable highlighting.
Right click to show the Highlighting pattern editor.
Filtering Left click to enable or disable filtering.
Right click to show the Filtering pattern editor.
Transforms Left click to enable or disable text transforms.
Right click to show the Text Transforms pattern editor.
Click Actions Left click to enable or disable click actions.
Right click to show the Click Actions pattern editor.
Find Dialog Shows the find dialog.
Bookmarks Shows the bookmarks dialog.
Select First Selects the first line in the log file causing the view to scroll to the top of the file.
Select Last Selects the last line in the log file causing the view to scroll to the end of the file.
Tail Causes the view to automatically scroll to the end as new data is loaded.
Watch File Enables or disables watching for file changes. If enabled, RyLogViewer will automatically reload the current log file whenever it is changed.
Assume Additive Enables or disables the assumption that all file changes are additive only. If enabled, RyLogViewer will only read new data added to the file when it detects that it has changed.
File Scroll Indicator
The file scroll indicator serves two purposes. Primarily, it shows where the currently displayed text is located within the log file as a whole. Secondarily, it is used to quickly navigate to another location within the file.

The different colours within the file scroll indicator convey the following information. The associated colours are set in the options dialog.
  • the portion of the file that is cached in memory,
  • the portion of the file that is currently being displayed,
  • the selected rows,
  • the bookmark locations.
Left clicking within the file scroll indicator selects the line at the corresponding position in the log file.
Text Encoding
RyLogViewer supports four common text encodings; ASCII, UTF-8, UCS-2 little-endian, and UCS-2 big-endian. The text encoding to use can be set under the Encoding menu. However, RyLogViewer also supports automatic detection of the file text encoding, so it is usually not necessary to set the text encoding explicitly.
Line Endings
There are three line ending conventions in common use today:
  • Windows/DOS - carriage return, line feed character pairs,
  • Unix - single line feed character,
  • Macintosh - single carriage return character,
RyLogViewer supports all three conventions and is able to automatically detect the line ending style in most cases. There is also support for custom line ending sequences that allow arbitrary combinations of characters to be interpreted as a line ending. A custom line ending can be set in the options dialog which can be accessed quickly by selecting the Custom option under the Line Ending menu.
Find Dialog / Searching
The ability to search for specific text within a log file is a vitally important feature. RyLogViewer is designed to make searching fast and simple, with convenient keyboard shortcuts that eliminate unnecessary swapping between the mouse and keyboard.
  • Access the find dialog using the 'Ctrl-F' key combination.
  • On opening, the search pattern text field has input focus so a pattern can be entered immediately.
  • The search pattern is initialised from the currently selected row.
  • Use the down arrow key to show and select previous search patterns.
  • The Enter key searches forward from the currently selected line, Control + Enter searches forward from the beginning of the log file.
  • Shift + Enter searches backward from the currently selected line, Control + Shift + Enter searches backward from the end of the log file.
  • Close the find dialog with the escape key.
In the main log view, the following keyboard shortcuts are also available:
  • F3 - Search forward from the currently selected line using the last search pattern
  • Shift+F3 - Search backward from the currently selected line using the last search pattern

Other Menu Options
Clear Log File
This option attempts to reset the contents of the loaded log file. It is not always possible for this operation to succeed as other applications may hold a write lock on the file. (Typically, the application writing to the log file does). If the Log Program Output feature is used, the program whose output is being logged may be holding the file lock. When file locks permit this operation however, it can be a handy feature.
