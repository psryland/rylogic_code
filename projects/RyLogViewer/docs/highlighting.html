Highlighting
Highlighting is one of the primary features of RyLogViewer. The purpose of highlighting is to make particular lines in the log file stand out based on their content. Many applications output data to log files and often important information can be hard to spot amongst the stream of regular output.
The image above labels each control in the highlighting pattern editor. A description of each control follows:
  1. Match Type
    Use these radio buttons to select the type of pattern you plan to create. The versatility, but also the complexity, of the patterns that can be created increases from left to right, with Substring being the simplest and Regular Expression being the most complex.
    • Substring
      This is the simplest pattern of all. The pattern matching algorithm looks for any occurrence of the sub string in each line of the log file. Those in which the sub string is found are considered a match.
    • Wildcard
      Wildcard patterns work in a similar way to Substring patterns. However, in a wildcard pattern the * and ? characters have special meaning. * means "match any number of characters", and ? means "match any single character".
