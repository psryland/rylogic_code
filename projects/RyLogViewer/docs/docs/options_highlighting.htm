Highlighting Tab
Highlighting is one of the primary features of RyLogViewer. The purpose of highlighting is to make particular lines in the log file stand out based on their content. Many applications output data to log files and often important information can be hard to spot amongst the stream of regular output.
  1. Pattern Editor
    The pattern editor is used to create and modify patterns that will match lines in the log data. The pattern editor is described in greater detail here.
  2. Active Patterns
    It is sometimes convenient to disable a pattern without deleting it. Highlighting patterns can be enabled or disabled by clicking in this column
  3. Expression
    This column displays the text of the highlighting pattern.
  4. Highlight Style
    The Full highlighting style means that the entire line in the log data is highlighted if the pattern matches any part of the line. The Partial style means only the matching part of the line is highlighted.
  5. Colours
    This column allows the highlighting colour scheme to be set for the pattern
  6. Highlight Pattern Editing
    Clicking the icon in this column loads the pattern into the pattern editor to be modified.
  7. Highlighting Pattern Sets
    Pattern sets are used to save and load groups of patterns. This is useful when RyLogViewer is used frequently for different types of logs. All of the highlighting patterns can be easily switched by selecting a pattern set from the drop down menu.
