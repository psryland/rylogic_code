Installation
System Requirements
  • Microsoft Windows XP or later (32 or 64 bit)
  • Microsoft .NET Framework 4.5.2
Normal Installation
RyLogViewer is typically installed via the application installer MSI file. With this type of installation, a folder is added to the user's Documents folder to contain the user's preferences, pattern sets, and licence information.
Portable Mode
RyLogViewer has been designed to support running from a removable drive. For this style of use, simply copy the 'RyLogViewer' directory to a removable/USB drive and create an empty file named portable (with no file extension) in the same directory as RyLogViewer.exe

On startup, RyLogViewer looks for the portable file in the same directory as the application executable. If found, the application starts in portable mode. Settings are loaded and saved to the same directory as the application. Alternatively, RyLogViewer can also be started in portable mode using the -p command line switch.

Note: To run RyLogViewer in portable mode (such as from a USB drive), the host system must have the .NET Framework 4.5.2 re-distributable package installed. This is typically already installed on most windows computers, however if needed, it can be downloaded from:
https://www.microsoft.com/en-us/download/details.aspx?id=42643
