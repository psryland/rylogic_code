Appearance Tab
Under the appearance tab you will find controls that affect how log data is displayed. Click on the yellow boxes in the image below to jump to a description of each control.
  1. Selection Colour
    Sets the colour scheme for selected rows.
  2. Log Text Colour
    Sets the colour scheme for non-selected text.
  3. Alternate Line Colours
    Check this option to have every alternate row displayed in a different colour to the main Log Text Colour.
  4. Alternate Line Colour Picker
    Sets the colour scheme for the alternate rows when alternate line colours are used.
  5. Row Height
    The height of each line displayed in the main log view
  6. Font
    Sets the font used to display the text in the main log view.
  7. Full Path in Title
    If checked, the full file path of the loaded log file is displayed in the application title bar
  8. Tab Size
    The size, in white space characters, to expand tab characters to.
  9. File Scroll Bar Width
    The vertical bar on the right side of the main window is the File Scroll Indicator. It is used to display the position of the currently cached portion of the log file, and to quickly navigate within the log file. This setting controls its visual width on the main log view.
  10. Cached Colour
    The colour used to represent the cached portion of the log file within the File Scroll Indicator.
  11. Visible Colour
    The colour used to represent the currently visible portion of the log file with the File Scroll Indicator
  12. Bookmark Colour
    The colour used to show bookmarked locations on the File Scroll Indicator also in the main log view.
