Plugins
The functionality of RyLogViewer can be extended via plugins implemented as .NET class libraries. Included with RyLogViewer are two managed DLLs RyLogViewer.Extensions.dll and Rylogic.dll. To create a plugin, create a .NET class library project and reference these two assemblies.
The RyLogViewer installation includes a directory called plugins.
