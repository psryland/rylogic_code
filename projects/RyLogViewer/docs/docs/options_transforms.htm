Transform Tab
Text transforms allow log data to be reformated before it is displayed. Reasons to transform log data include; replacing error codes with human readable strings, simplifying log lines to increase readability, and more. Transforms are also useful when used with the Export feature, for reformatting large files.
  1. Transform Editor
    The transform editor is used to create and modify patterns that will match lines in the log data and apply text transformations to them. The transform editor is described in greater detail here.
  2. Active Transforms
    It is sometimes convenient to disable a transform without deleting it. Transform patterns can be enabled or disabled by clicking in this column
  3. Expression
    This column displays the text of the transform pattern.
  4. Transform Pattern Editing
    Clicking the icon in this column loads the transform into the transform editor to be modified.
  5. Transform Pattern Sets
    Pattern sets are used to save and load groups of patterns. This is useful when RyLogViewer is used frequently for different types of logs. All of the transform patterns can be easily switched by selecting a pattern set from the drop down menu.
