F.A.Q.
Other programs using adb.exe disconnect when running RyLogViewer
Check whether you have more than one instance of adb.exe installed on your system and verify that RyLogViewer is running the same version of adb.exe as the other program. Running different versions of adb.exe can cause the ADB service to reset which is usually the cause of existing adb.exe instances disconnecting.

To find out which version of adb.exe RyLogViewer is running, open the Android Logcat wizard.

RyLogViewer's use of adb.exe is no different to running adb.exe from a command prompt, so to help diagnose issues like this, trying running adb.exe from a command prompt window.
