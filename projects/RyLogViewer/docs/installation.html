Installation
System Requirements
  • Microsoft Windows XP or later (32 or 64 bit)
  • Microsoft .NET Framework 4.0 Client Profile
Portable Mode
RyLogViewer has been designed to support running from a user directory or removable drive. On startup, RyLogViewer looks for a file in the same directory as the application with the name portable. If found, the application starts in portable mode where settings are loaded and saved to the same directory as the application. Alternatively, RyLogViewer can also be started in portable mode using the -p command line switch.

Note: To run RyLogViewer in portable mode (such as from a USB drive), the host system must have the .NET Framework 4.0 client profile redistributable package installed. This can be downloaded from:
http://www.microsoft.com/en-nz/download/details.aspx?id=24872
