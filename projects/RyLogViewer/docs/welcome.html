RyLogViewer is an application designed to make the viewing of log files or streaming log data easier through the use of highlighting, filtering, and text transformations. RyLogViewer can display log files of unlimited size[1], as well as files that are being changed in real-time, much like the Unix 'tail' command.

RyLogViewer can also receive and display log data from a wide variety of sources including program standard output, serial port connections, data streamed over a network, and named pipes.

RyLogViewer is a .NET Framework 4.0 Client Profile application designed to run on Microsoft Windows versions XP and later, both 32 and 64bit.
[1] limited only by the operating system.
