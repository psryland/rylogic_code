Appearance Tab
Under the appearance tab you will find controls that affect how log data is displayed. Click on the yellow boxes in the image below to jump to a description of each control.
  1. Selection Colour
    Sets the text and background colours of selected rows in the main log view.
  2. Log Text Colour
    Sets the text and background colours used by text in the main log view.
  3. Alternate Line Colours
    Check this option to have every alternate row display in a different colour to the main Log Text Colour. The colour selection area ([4]) is used to set the colour used.
  4. Row Height
    This value corresponds to the height of each line displayed in the main log view
  5. Font
    Sets the font used to display the text in the main log view.
  6. File Scroll Bar Width
    The vertical bar on the right side of the main window is the File Scroll Indicator. It is used to display the position of the currently cached portion of the log file, and to quickly navigate within the log file. This setting controls its visual width on the main log view.
  7. Cached Colour
    This control changes the colour of the cached file range indicator within the File Scroll Indicator.
  8. Visible Colour
    This control changes the colour of the currently visible file range indicator with the File Scroll Indicator
  9. Bookmark Colour
    This control changes the colour used to show bookmarked locations on the File Scroll Indicator also in the main log view.
