RyLogViewer is designed to make the viewing of log files or streaming log data easier through the use of highlighting, filtering, and text transformations. RyLogViewer can display log files of unlimited size (limited only by the operating system), as well as files that are being changed in real-time (much like the unix 'tail' command).
RyLogViewer is a .NET Framework 4.0 application designed to run on Microsoft Windows versions XP and later, both 32 and 64bit.
