PR C# Assembly


This C# assembly contains a collection of miscellaneous controls and helper classes.

  • View3D.dll C# interface
  • View Image Control
  • View Video Control using DirectShow.NET
  • Interactive Graph Control
  • VT100 terminal emulator
  • Helper Extensions for System.Action and System.EventHandler
  • Sortable Binding List
  • Generic Ring Buffer
  • Generic Type Converter for displaying types in a PropertyGrid
  • String Attribute for attaching strings to enums
  • Units Attribute
  • Weak Delegates
  • Extensions to DataGridView for cut/copy/paste functionality
  • 'Hover' Scrolling - mouse wheel scrolling of control under the mouse pointer, not necessarily with input focus
  • Progress dialog with cancel option encapsulating a background worker
  • Recent files menu item helper
  • CSV file loader/saver
  • XML helper methods
  • Win32 API wrapper
  • Base64 encoder/decoder
