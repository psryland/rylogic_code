Imager


Imager is a multimedia viewer and slide show program based on directx and directshow. It can display images, videos, and audio clips all within the one application and has a simple interface for cycling through media files manually or as a slide show. It can also be used as a screen saver for displaying your photo and video collection while your PC is idle.

User Manual:

At the core of Imager is the media file list which is viewable through the Window menu or by pressing F8. This list can be generated a number of ways:


  • File Menu -> Open File option
  • File Menu -> Directories option
  • Drag 'n' Drop of files and/or folders into the main application window
  • Drag 'n' Drop of files and/or folders into the Media List window

Open File Menu Option

The Open File menu option allows multiple files to be selected from a standard open file dialog. The selected files are added to the media file list.


Directories Menu Option

The Directories menu option opens a dialog allowing the user to manage a collection of folders. When OK is selected, the listed folders are searched for media files and any that are found are added to the media list.
