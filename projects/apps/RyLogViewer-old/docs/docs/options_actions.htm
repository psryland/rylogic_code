Actions Tab
Actions allow a program or script to be started when a line in the log data is double clicked with the mouse. The action can use data extracted from the line that was clicked in the command. A typical use case for this feature is to load data mentioned in the log in an external application.
  1. Pattern Editor
    The pattern editor is used to create and modify patterns that will match lines in the log data. The pattern editor is described in greater detail here.
  2. Active Patterns
    It is sometimes convenient to disable a pattern without deleting it. Action patterns can be enabled or disabled by clicking in this column
  3. Expression
    This column displays the text of the action pattern.
  4. Action Command
    The command to execute when the action pattern matches. This command can start another application or run a script using data extracted via capture groups from the selected log line. Click in the column to edit the command for the action.
  5. Action Pattern Editing
    Clicking the icon in this column loads the pattern into the pattern editor to be modified.
  6. Action Pattern Sets
    Pattern sets are used to save and load groups of patterns. This is useful when RyLogViewer is used frequently for different types of logs. All of the action patterns can be easily switched by selecting a pattern set from the drop down menu.
