Main Window
  1. Log View
    This region displays the contents of the loaded or streamed log data. Scroll this view using the mouse wheel, page up/down keys, or up/down arrow keys. Select rows by holding down the left mouse button and dragging, or hold down the Shift or Control key while selecting. (Note: due to the way RyLogViewer spools data it is not possible to select and copy the entire contents of very large files. Only the portion of the file currently loaded is selected and copied.)
    Right clicking in this view displays a context menu. The menu items are:
    Option Function
    Select All Select all log file data that is currently loaded into memory. For large files this may not be the entire file contents due to the way RyLogViewer spools log data.
    Copy Copy the selected rows to the system clip board.
    Clear Log Attempts[1] to reset the contents of the log file.
    Highlight Row
    Filter Row
    Transform Row
    Action Row
    		 Opens the highlighting, filtering, text transforms, or click actions pattern editor, initialised with data from the selected line.
    Find Next
    Find Previous
    		 Jump to the next or previous line in the log file that matches the last search pattern.
    Toggle Bookmark Add or remove a bookmark from the selected line.
    [1] Clearing the log file can fail if the application generating the log file does not allow write access to the file.
  2. Tool Bar
    The tool bar contains buttons that allow quick navigation to, and control of, various features of RyLogViewer. In order from left to right, these are:
    Button Function
    Open File Choose a log file to be displayed.
    Refresh Update the display by forcing a reload of data from the currently loaded log file.
    Quick Filter Only show rows that are highlighted.
    Highlighting Left click to enable or disable highlighting.
    Right click to show the Highlighting pattern editor.
    Filtering Left click to enable or disable filtering.
    Right click to show the Filtering pattern editor.
    Transforms Left click to enable or disable text transforms.
    Right click to show the Text Transforms pattern editor.
    Click Actions Left click to enable or disable click actions.
    Right click to show the Click Actions pattern editor.
    Find Dialog Shows the find dialog.
    Bookmarks Shows the bookmarks dialog.
    Select First Selects the first line in the log file causing the view to scroll to the top.
    Select Last Selects the last line in the log file causing the view to scroll to the end.
    Tail Causes the view to automatically scroll to the end as new data is loaded.
    Watch File Enables or disables watching for file changes. If enabled, RyLogViewer will automatically display the new log data whenever it changes.
    Assume Additive Enables or disables the assumption that all file changes are additive only. If enabled, RyLogViewer will only read new data appended to the end of the file when it detects that the file has changed. Otherwise, all data around the current selection position is reloaded.
  3. File Scroll Indicator
    The file scroll indicator serves two purposes. Primarily, it shows where the currently displayed data is located within the log file as a whole. Secondarily, it is used to quickly navigate to another location within the file.

    The different colours within the file scroll indicator convey the following information. These are the default colours, however the file scroll indicator colours can be changed in the options dialog.
    • white - the regions of the log data not currently loaded into memory. This only occurs for large log files that cannot be fully cached in memory.
    • pale blue - the portion of the log data that is cached in memory.
    • light blue - the portion of the file that is currently visible in the main window.
    • green - the selected rows.
    • light purple - the bookmark locations.
    Left clicking within the file scroll indicator selects the line at the corresponding position in the log file.

    Note: When a small log file is loaded you may wonder why the selection indication on the file scroll indicator changes size as different lines are selected. This is because the indicator represents the byte range of the selected line within the file, longer lines correspond to larger byte ranges and therefore a larger selection indicator.
  4. Text Encoding
    RyLogViewer supports four common text encodings; ASCII, UTF-8, UCS-2 little-endian, and UCS-2 big-endian. The text encoding to use can be set under the Encoding menu. RyLogViewer supports automatic detection of the file text encoding, however, so it is usually not necessary to set the text encoding explicitly.
  5. Line Endings
    There are three line ending conventions in common use today:
    • Windows/DOS - carriage return, line feed character pairs,
    • Unix - single line feed character,
    • Macintosh - single carriage return character,
    RyLogViewer supports all three conventions and is able to automatically detect the line ending style in most cases. There is also support for custom line ending sequences that allow arbitrary combinations of characters to be interpreted as a line ending. A custom line ending can be set in the options window which can be accessed quickly by selecting the Custom option under the Line Ending menu.
  6. Find Dialog / Searching
    The ability to search for specific text within a log file is a vitally important feature. RyLogViewer is designed to make searching fast and simple, with convenient keyboard shortcuts that eliminate unnecessary switching between the mouse and keyboard. The following list details the shortcuts available:
    • Access the find dialog using the Control+F key combination.
    • On opening, the search pattern combo box is selected so that a pattern can be entered immediately.
    • The search pattern is initialised from the currently selected row in the log file.
    • Use the down arrow key to show and select previous search patterns.
    • Hit Enter to search forward from the currently selected line,
      Control+Enter searches forward from the beginning of the log file.
    • Hit Shift+Enter to search backward from the currently selected line,
      Control+Shift+Enter searches backward from the end of the log file.
    • Close the find dialog with the Escape key.
    In the main log view, the following keyboard shortcuts are also available:
    • F3 - Search forward from the currently selected line using the last search pattern,
    • Shift+F3 - Search backward from the currently selected line using the last search pattern,
    • Control+F3 - Set the search pattern to the contents of the currently selected line, and then search forward for the next occurrence.
    • Control+Shift+F3 - Set the search pattern to the contents of the currently selected line, and then search backward for the previous occurrence.
    The find feature is based on the same algorithms as the other pattern matching features in RyLogViewer (highlighting, filtering, etc). The controls function in the same way as described in the pattern editor section, otherwise:
    1. Drop down list of previous search patterns.
    2. A list of previous search patterns.
    3. Find Next - searches forward from the currently selected line.
    4. Find Previous - searches backward from the currently selected line.
    5. Bookmark All - sets a bookmark on all lines that match the search pattern.
    When a search reaches the start or end of the log file, a message is displayed in the Status bar that indicates that the Start of File, or End of File has been reached.
  7. Bookmarks
    Bookmarks provide a means of saving and returning to particular locations in the log file. Like the Find feature, convenient keyboard shortcuts are available to eliminate unnecessary switching between the mouse and keyboard.
    • Control+F2 - Creates or clears a bookmark from the currently selected line.
    • F2 - Jumps forward through the file to the next bookmark, looping around to the beginning of the file when the end of the file is reached.
    • Shift+F2 - Jumps backward through the file to the previous bookmark, looping around to the end of the file when the start of the file is reached.
    • Access the bookmarks dialog using the Control+B key combination.
    The bookmarks dialog is simply a list of saved bookmark positions. Selecting a bookmark in the list causes the main view to jump to the bookmarked position.
  8. Status
    The status bar displays information about the currently loaded log file, as well as transient informational messages. From left to right, the status bar shows:
    • The byte offset of the currently selected line into the log file,
    • The line ending mode being used to parse lines in the log file,
    • The text encoding being used to read text from the log file,
    • The file loading progress bar,
    • A transient message describing the result of a recent operation.
    • A static message describing the state of a running operation.
Tools Menu
Always On Top
When enabled, causes the RyLogViewer window to stay above other currently open windows. This mode is helpful when monitoring log output while working with other programs.
Monitor Mode
As an extension of the Always On Top option, Monitor Mode allows RyLogViewer to be put into a state where the main window frame, menu, tool bar, and file scroll indicator are all hidden, maximising the area of the main log view. The window can also optionally be made semi-transparent, and set to 'click-through' mode where keyboard and mouse events pass through to any windows below. To exit Monitor Mode click on the RyLogViewer icon in the system tray.
Clear Log File
This option attempts to reset the contents of the loaded log file. It is not always possible for this operation to succeed as other applications may hold a lock on the file preventing write access. (Typically, the application writing to the log file does this). When file locks permit this operation however, it can be a handy feature.
Options Dialog
This item opens the application options dialog.
Help Menu
View Help
Opens the main application documentation. Documentation is in HTML format, so selecting this option launches the default web browser.
First-Run Tutorial
The first-run tutorial is a wizard-style walk through of the main features of RyLogViewer. It highlights the various aspects of the application and provides a short description of usage.
Tip of the Day
Tip-of-the-Day messages are typically displayed at startup and are used to convey features of RyLogViewer that the user may be unaware of. They are brief descriptions of a feature, often with links to more information provided in the main documentation.
Activate Licence
Opens a dialog that displays the current licence information. After purchasing a licence, RyLogViewer can be activated via this menu option.
