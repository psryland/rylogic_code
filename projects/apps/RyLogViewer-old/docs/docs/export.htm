Export
The export option (available from the File menu) is used to write some or all of the current log data to a file. Filtering and transforms can be optionally applied, making RyLogViewer a powerful tool for converting, reformatting, and filtering large files.
  1. Output File
    The location and name of the file to create by exporting the log data.
  2. Export Range
    The log data to include in the export.
    • Whole File - All log data is exported.
    • Current Selection - Only selected lines from the log data are exported. Note: since RyLogViewer only loads parts of large log data files into memory, selecting All and exporting will only export the portion of the file loaded into memory.
    • Specific Range - Exports data within the given file position range. The range is specified in bytes rather than lines because the line number is not generally known. The file position in the Status bar can be used to determine appropriate range values. Note, the start of the range is adjusted to include the start of the line that it falls on.
  3. Line Ending
    The characters that signify the end of a line in the log data. These are typically the same as the values used in the main log view but may be changed without modifying the main settings.
  4. Column Delimiter
    The characters that signify the boundaries between columns. These are typically the same as the values used in the main log view but may be changed without modifying the main settings.
