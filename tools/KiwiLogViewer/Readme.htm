Product Overview

Kiwi Log Viewer displays text based log files in a tabular format. Only a small section of the file is read from disk at a time which saves memory and allows you to view a file that would be too big to fit in memory.

The tail option monitors the specified log file for changes and displays any new data that is added in real time.

Text string matching allows important text to be highlighted by using colors or bold/italic font emphasis. Text matching options include simple sub-string matching or more powerful PERL style Regular Expression matching.

Kiwi Log Viewer will work on the following operating systems:

  • Windows 2000
  • Windows XP
  • Windows 2003
  • Windows Vista
  • Windows 2008

Features

Features available in the Free edition:

Running Kiwi Log Viewer in free mode allows you to use it for as long as you like and it will never expire.

  • Open recent files (2 files)

  • Importing/Exporting of highlights

  • Returns up to 100 matches of filtered results
  • Monitor log files for new data and display in real time

  • Copy selected rows to the clipboard in tab delimited format
