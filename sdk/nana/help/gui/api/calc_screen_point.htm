nana::gui::API::calc_screen_point

Description
Converts a window coordinates of a specified point to the screen coordinates.

Prototype

bool calc_screen_point(nana::gui::window wnd, nana::point& pos);

Parameters

wnd:  a handle to the window whose point is used for the conversion.
pos:  specifes a window coordinates of the point to be converted. And the new screen coordinates are copied into pos if function succeeds.

Return Value

If the function succeeds, it returns true, otherwise it returns false.

Remarks

None.
