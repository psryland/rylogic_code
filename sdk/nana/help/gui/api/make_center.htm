nana::gui::API::make_center

Description
Retrieves a rectangle which is in the center of screen.

Prototype

nana::rectangle make_center(unsigned width, unsigned height);
Parameters

width:  Specifies the width of the rectangle.
height:  Specifies the height of the rectangle.

Return Value

It returns a rectange with the specifies the width and height, and the rectangle is position in the center of the screen.

Remarks

None.
