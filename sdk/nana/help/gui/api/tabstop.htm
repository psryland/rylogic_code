nana::gui::API::tabstop

Description
Sets a window that owns the tabstop.

Prototype

void tabstop(nana::gui::window wnd);

Parameters

wnd:  a handle to the window which is to be set.

Return Value

None.

Remarks

If a window owns the tabstop, the window can get keyboard focus while pressing the Tab key to switch the focus.

See also

eat_tabstop, move_tabstop
