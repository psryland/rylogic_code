nana::gui::API::move_tabstop

Description
Sets the focus to the window which tabstop is near by the specified window.

Prototype

nana::gui::window move_tabstop(nana::gui::window wnd, bool next);

Parameters

wnd:  A handle to the window which the focus is moved from it.
next: Move the focus to the next tabstop window or the previous tabstop window.

Return Value

A window handle to the new focus window indicates success.
0 indicates failure. If there is noly one tabstop window, 0 indicates no change to the focus.

Remarks

None.

See also
