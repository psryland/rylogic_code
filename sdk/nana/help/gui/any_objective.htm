Any Objective

Description
The Any Objective is an object that may attach to some other object.

Model of
Concept

Public base classes

None

Template Parameter

typename IndexType The type of index
int Dimension The dimension

Typedef

anyobj_index_t The type of index. It is available if Dimension is greater than 0.

Members

template<typename Target>
void anyobj(anyobj_index_t ..., Target&)		 Attaches to an object. The number of index is determinated by Dimension.
template<typename Target>
Target * anyobj(anyobj_index_t ...) const		 Retrieves the attached object. It returns a null pointer if it is empty or the specified type is not matched.

C++11 Members

template<typename Target>
void anyobj(anyobj_index_t ..., Target &&)		 Ditto.
