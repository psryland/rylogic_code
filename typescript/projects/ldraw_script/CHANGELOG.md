# Change Log

All notable changes to the "ldraw-script" extension will be documented in this file.

## [1.1.0] - 2026-02-05

### Added

- IntelliSense auto-completion for LDraw script keywords
- Context-aware completions (only valid keywords shown based on cursor position)
- Template syntax displayed in completion popup
- Build-time parser for `ldraw_templates.cpp`

## [1.0.0]

### Added

- Initial release
- Syntax highlighting for LDraw script
